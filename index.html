Javascript Basico

Antes de comenzar, es importante saber que JavaScript es un lenguaje de programación muy popular que se utiliza en la web para crear páginas web interactivas y dinámicas. Fue creado en 1995 por Brendan Eich mientras trabajaba en Netscape.

Una de las características principales de JavaScript es que se ejecuta en el lado del cliente, lo que significa que se ejecuta en el navegador del usuario y no en el servidor. Se utiliza para agregar interactividad y dinamismo a los sitios web, permitiendo a los usuarios interactuar con los elementos de la página, realizar acciones y ver resultados en tiempo real.

En el lado del cliente, JavaScript se utiliza para validar formularios, crear efectos visuales, animar elementos de la página, crear botones que realicen una acción al hacer clic, entre otros. También se utiliza en el lado del servidor, mediante plataformas como Node.js, para desarrollar aplicaciones web y servicios web.

JavaScript es un lenguaje de programación orientado a objetos, interpretado y de alto nivel. Es muy versátil y se integra bien con otros lenguajes y tecnologías. Es compatible con HTML y CSS, lo que significa que se puede utilizar para interactuar con la estructura y el estilo de una página web.

En resumen, JavaScript es un lenguaje de programación muy importante para el desarrollo web. Es utilizado tanto en el lado del cliente como del servidor, y es capaz de agregar interactividad y dinamismo a las páginas web, lo que lo convierte en una herramienta fundamental para cualquier desarrollador web.

Parte 2: Variables y tipos de datos en JavaScript

JavaScript utiliza variables para almacenar y manipular datos. En esta sección, aprenderás cómo declarar variables y los tipos de datos que puedes utilizar.

1. En la consola del navegador, escribe el siguiente código:

let nombre = "Juan";
let edad = 25;
let casado = false;
console.log(nombre);
console.log(edad);
console.log(casado);

2. Haz clic en "Enter" para ejecutar el código. Deberías ver el valor de la variable nombre (que es "Juan"), el valor de la variable edad (que es 25) y el valor de la variable casado (que es false).

Explicación: En este código, hemos declarado tres variables. La variable nombre es una cadena de caracteres (también conocida como "string" en inglés), la variable edad es un número entero (también conocido como "integer" en inglés) y la variable casado es un valor booleano (es decir, true o false).

Parte 3: Operadores en JavaScript

JavaScript utiliza operadores para realizar operaciones en datos. En esta sección, aprenderás los operadores básicos que puedes utilizar.

1. En la consola del navegador, escribe el siguiente código:

let x = 10;
let y = 5;
console.log(x + y);
console.log(x - y);
console.log(x * y);
console.log(x / y);

2. Haz clic en "Enter" para ejecutar el código. Deberías ver la suma, resta, multiplicación y división de las variables x e y.

Explicación: En este código, estamos utilizando los operadores aritméticos +,-,* y / para realizar operaciones matemáticas en las variables x e y.

Parte 4: Condicionales en JavaScript

JavaScript utiliza condicionales para tomar decisiones basadas en condiciones. En esta sección, aprenderás cómo utilizar la declaración if en JavaScript.

1. En la consola del navegador, escribe el siguiente código:

let edad = 18;
if (edad >= 18) {
console.log("Eres mayor de edad");
} else {
console.log("Eres menor de edad");
}

2. Haz clic en "Enter" para ejecutar el código. Deberías ver el mensaje "Eres mayor de edad".

Explicación: En este código, estamos utilizando la declaración if para verificar si la variable edad es mayor o igual a 18. Si es así, se ejecutará el primer bloque de código (que imprimirá "Eres mayor de edad"). Si no es así, se ejecutará el segundo bloque de código (que imprimirá "Eres menor de edad").

Parte 5: Bucles en JavaScript

JavaScript utiliza bucles para repetir bloques de código. En esta sección, aprenderás cómo utilizar el bucle for en JavaScript. 1. En la consola del navegador, escribe el siguiente código:

for (let i = 0; i < 5; i++) {
console.log(i);
}

2. Haz clic en "Enter" para ejecutar el código. Deberías ver los números del 0 al 4 en la consola.

Explicación: En este código, estamos utilizando el bucle for para imprimir los números del 0 al 4 en la consola. La variable i comienza en 0, se repite mientras i sea menor que 5 y se incrementa en 1 en cada iteración del bucle.

Parte 6: Funciones en JavaScript

JavaScript utiliza funciones para agrupar bloques de código que realizan una tarea específica.
En esta sección, aprenderás cómo definir y llamar funciones en JavaScript. 1. En la consola del navegador, escribe el siguiente código:

function suma(a, b) {
return a + b;
}
console.log(suma(2, 3));

2. Haz clic en "Enter" para ejecutar el código. Deberías ver el número 5 en la consola.

Explicación: En este código, estamos definiendo una función llamada suma que toma dos parámetros (a y b) y devuelve la suma de esos dos valores. Luego, estamos llamando a la función suma con los argumentos 2 y 3, y estamos imprimiendo el resultado en la consola.

Parte 7: Objetos en JavaScript

JavaScript utiliza objetos para representar datos estructurados. En esta sección, aprenderás cómo definir y acceder a objetos en JavaScript.

1. En la consola del navegador, escribe el siguiente código:

let persona = {
nombre: "Juan",
edad: 25,
direccion: {
calle: "Calle 123",
ciudad: "Bogotá"
}
};
console.log(persona.nombre);
console.log(persona.edad);
console.log(persona.direccion.ciudad);

2. Haz clic en "Enter" para ejecutar el código. Deberías ver el nombre, la edad y la ciudad de la persona en la consola.

Explicación: En este código, estamos definiendo un objeto llamado persona con tres propiedades (nombre, edad y direccion). La propiedad direccion es a su vez otro objeto con dos propiedades (calle y ciudad). Luego, estamos accediendo a las propiedades del objeto persona utilizando la notación de punto.
